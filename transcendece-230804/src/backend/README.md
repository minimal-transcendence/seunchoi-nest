## Description

Backend repository for [Mini-transcendence] project

## Running the app

```bash
# watch mode
$ npm run start:migrate:dev

# production mode
$ npm run start:migrate:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

## Api
|Endpoint|Request|Response|Param|Args|
|--------|-------|--------|-----|----|
|/user|GET|User[]|||
|/user/:id|GET|User|id||
|/user/:id|Patch|User|id|score, etc|
|/user/:id/friend|GET|int[]|userid||
|/user/:id/friend|PATCH|int[]|userid|isAdd: boolean\nfriend: number|
|/user/:id/matchhistory|GET|Matchhistory[]|userid||
|/matchhistory|GET|Matchhistory[]|||
|/matchhistory|POST|Matchhistory[]||winnerId: number\nloserId: number|
